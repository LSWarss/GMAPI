GMAPI - Gaming Premiers API

This API gives you release dates, developer data and much more of your favorite games.

GET
https://gmapi.org/games

Returns list of all games.

GET
https://gmapi.org/games/platform/PC

Returns list of all games for given platform. For instance PC.

GET
https://gmapi.org/games/platforms

Returns list of all platforms.

GET
https://gmapi.org/games/genres

Returns list of all genres.

POST
https://gmapi.org/games/scrape

Starts out manual scraping of data. NEEDS TOKEN PROVIDED BY SERVICE OWNER.
